# Adidas_Dashboard
## 📊 Adidas Sales Dashboard – Power BI Project

This project is an **interactive Adidas Sales Dashboard** built using **Power BI** to analyze sales performance and generate actionable business insights. The dashboard combines strong **data visualization, storytelling, and user-friendly design** to help stakeholders make informed decisions.

It is designed as a **5-page interactive report**, allowing users to explore data across multiple dimensions such as sales trends, product performance, regional analysis, customer behavior, and profitability.

---

## 🚀 Key Highlights

* 📈 Dynamic **KPI tracking** (Sales, Profit, Orders, Growth)
* 📊 Interactive visuals including **line charts, bar charts, and donut charts**
* 🎯 Built-in **insight sections** for quick business understanding
* 🎛️ Fully functional **slicers** (Date, Region, Product Category, Retailer)
* 🔄 Smooth **navigation across 5 dashboard pages**
* 🎨 Clean and consistent **Adidas-themed UI (Red, Black, White)**

---

## 🧠 Business Value

This dashboard helps in:

* Identifying **top-performing regions, products, and retailers**
* Tracking **sales growth and seasonal trends**
* Analyzing **profitability and business performance**
* Understanding **customer and regional distribution**
* Supporting **data-driven decision making**
